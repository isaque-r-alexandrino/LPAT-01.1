**Calculadora em Bash**
Uma calculadora simples desenvolvida em Bash para operações matemáticas básicas via terminal.

**Funcionalidades**
- Soma: Adição de dois números
- Subtração: Subtração de dois números
- Multiplicação: Multiplicação de dois números
- Divisão: Divisão de dois números com tratamento para divisão por zero

**Pré-requisitos**
- Sistema Unix/Linux ou WSL no Windows
- Bash shell
- Pacote bc instalado (calculadora de precisão arbitrária)
